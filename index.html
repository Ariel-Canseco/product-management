🛍️ Gestor de Productos

Descripción del Proyecto

Esta aplicación es un sistema CRUD (Crear, Leer, Actualizar y Eliminar) desarrollado con HTML, CSS y JavaScript, cuyo propósito es gestionar un conjunto de productos de manera dinámica.

Permite a los usuarios agregar nuevos productos, visualizarlos en una lista, editarlos, eliminarlos, así como filtrarlos mediante un buscador y rangos de precio. Además, utiliza localStorage para almacenar la información de forma persistente en el navegador.

Este proyecto tiene como objetivo reforzar el uso del DOM, la manipulación de datos en el navegador y la aplicación del paradigma de Programación Orientada a Objetos (POO) en JavaScript.
